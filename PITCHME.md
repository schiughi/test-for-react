### スナップショットテスト


+++

#### スナップショットテストって？

+ UI が予期せず変更されていないかを確認するテスト | 

+ 最初に実行したテストのアウトプットが保存され、二回目以降のテストは以前のスナップショットとの比較して一致していれば通過する。 |


+++

#### 例

+ 開発者「`<LinkToHome />`というコンポーネントのURLを変更しました!」

+++

#### コード

```javascript
it('renders correctly', () => {
  const tree = renderer
    .create(<LinktoHome />)
    .toJSON();
  expect(tree).toMatchSnapshot();
});
```
